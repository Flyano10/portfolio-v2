INSTRUKSI MENAMBAHKAN GAMBAR PROJECT:

1. Letakkan gambar project Anda di folder ini (public/images/)

2. Nama file gambar harus sesuai dengan yang ada di Projects.tsx:
   - nstore.jpg (untuk project NSTORE)
   - office-management.jpg (untuk project Office Management System)

3. Format gambar yang didukung: JPG, JPEG, PNG, WEBP

4. Ukuran gambar disarankan: 
   - Lebar: minimal 800px
   - Rasio: 16:9 atau 4:3
   - Format: Landscape (mendatar)
