# Portfolio Website

Personal portfolio website built with Next.js 16, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Features

- Responsive design
- Dark mode support
- Modern UI/UX
- Project showcase with image gallery
- Contact form
- SEO optimized

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```

## Project Structure

```
src/
  app/          # Next.js app router pages
  components/   # React components
public/
  images/       # Project images and assets
```

## Deployment

Deploy to Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically
